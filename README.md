# Web-Page-of-Cancer-Awareness-Support

Description:

I have created a simple, responsive web page on “Cancer Awareness & Support” that includes:

A landing page with a short awareness message and a banner image

A contact form (Name, Email, Message) — no backend

A real-time quotes section fetched from a public API

Fully responsive layout built using HTML, CSS, and JavaScript

Tech Stack:

HTML5 – Page structure

CSS3 – Styling and responsiveness

JavaScript (Fetch API) – Displaying real-time quotes

GitHub Pages – Hosting
